aml-crypto-test is a test project developed using React, Typescrypt, D3.js, RTK, MUI, SCSS.

It allows you to get data with the node.id and display it as a graph is the D3-simulation.
By clicking on existing nodes you can see it's info.

To run the project on localhost:
1. run test server that goes with the task 
2. npm install
3. npm start
